# yoongsim-bot

Telegram bot for ugly cryptocurrency trader
[main.py](main.py)
Automatically send new positions and close positions status


## Requirements

```
pip install -r requirements.txt
```

## .env
```
TELEGRAM_API_TOKEN=
TELEGRAM_CHAT_ID=
BINANCE_API_KEY=
BINANCE_API_SECRET=
TRADER_NAME=
```

## Usage

```
python main.py
```

## Command

```
/ping
/positions
```
